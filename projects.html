Face Recognition
The Face Recognition process runs as a Python script that recognizes specific objects in static images. The algorithm utilizes a histogram of gradient orientations as a template to detect an object pattern.
I assembled the templates on the right by collecting the histogram of gradient orientations on a face/traffic sign image database.
Vaccine Dash (Videogame)
Vaccine Dash was a team project game for my Game Design course. I contributed to the source code in the Scratch block code language. I focused on designing the levels and artistic parts of the game. I drew each map level and produced the in-game sound score. I also wrote the main story of the game to increase character connection in the game.
Check out the website
The game takes place in a hospital during an apocalyptic pandemic. I used Canva and Pixlr to sketch the game environment. For each level, I focused on increasing the difficulty by adding obstacles and other horror-mystery aesthetics.

Tinder Map (App)
Tinder Map is an app prototype developed and designed by me. It allows users to find nearby places via swiping. The app shows a picture with a description of a nearby place for the user to swipe right (save it for later) or left (discard it). This app facilitates looking for places nearby.
I used the Marvel App website to create the prototype.

ZotFeeds (App)
ZotFeeds is an app prototype developed as a group project. ZotFeeds connects food donors to drivers, and drivers to food banks. Drivers would pick up food and bring it to food banks requesting food donations. Food donors can be restaurants or supermarkets. This app facilitates food donations by creating a system to easily donate food.
I wrote the pseudo-code for the app and the design of the database.
